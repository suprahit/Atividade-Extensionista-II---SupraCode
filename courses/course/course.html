Introdução ao Desenvolvimento Web

Fundamentos essenciais para começar a criar sites e aplicações web.

Sobre o Curso

Este curso é projetado para iniciantes que desejam aprender os fundamentos do desenvolvimento web. Através de uma abordagem prática, você aprenderá a construir sites e aplicações responsivas usando HTML, CSS e JavaScript.

Detalhes do Curso

  • Duração: 8 semanas
  • Carga Horária: 24 horas
  • Pré-requisitos: Nenhum
  • Certificação: Sim

Cronograma

  • Semana 1: Introdução ao HTML
  • Semana 2: Estilização com CSS
  • Semana 3: Layouts Responsivos
  • Semana 4: Fundamentos de JavaScript
  • Semana 5: Manipulação DOM
  • Semana 6: Projetos Práticos
  • Semana 7: Melhoria de Projeto
  • Semana 8: Revisão e Avaliação
